
base.py  - Настройки общие для всех экземпляров проект

local.py - Это файл настроек, который вы используете, когда вы работаете над проектом локально. 
           Ло-Калиброванные настройки, специфичные для разработки, включают Режим DEBUG, уровень 
           журнала и активация инструменты разработчика, такие как django-debug-toolbar.

staging.py Постановочная версия для запуска полуприватного версия сайта на рабочем сервере. 
           Здесь менеджеры и клиенты должны поищите, прежде чем ваша работа будет перемещена в производство.

test.py    Настройки для запуска тестов, включая test бегуны, определения базы данных в памяти, и настройки журнала.

production.py Это файл настроек, используемый вашим живым рабочий сервер(ы). 
              То есть сервер(ы) которые размещают настоящий живой веб-сайт. 
              Этот файл содержит содержит только настройки производственного уровня. Это иногда называется prod.py.


ЧТОБЫ ЗАПУСТИТЬ 

Настроить  python manage.py shell --settings=config.settings.local

Запустить  python manage.py runserver --settings=config.settings.local
